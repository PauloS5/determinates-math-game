Conceitos

Matrizes

Matriz é um quadro retângular composto por números. Uma matriz costuma ser representada por uma letra maiúscula, tal como A, e tem um determinado número de linhas (m) e de colunas (n). As matrizes tem que ser representadas por letras maiúsculas, onde os elementos da matriz são representados por letras minúsculas. Matrizes são usadas muito na área da computação, desenvolvimento de Softwares, problemas em matemática, criação de tabelas ordenadas, como no caso do programa Excel.

Determinantes

Determinante é uma função matricial que associa a cada matriz quadrada um escalar, ou seja, é uma função que transforma uma matriz quadrada em um número real. Uma determinante de uma matriz pode ser encontrada subtraindo a diagonal principal pela diagonal secundária da matriz, existem dois jeitos comumente usados para encontrar o determinante de uma matriz: A regra de Sarrus, e o teorema de Laplace. Sabendo determinantes, podemos entender outros conceitos, por exemplo: Se uma matriz é inversa ou não e se o sistema linear tem solução e é única.

Sistemas Lineares

Sistema Linear é um conjunto finito de equações lineares aplicadas num mesmo conjunto, igualmente finito, de variáveis. Um sistema linear pode ser possível e determinado, tendo somente uma solução, pode ser possível e inderteminado, tendo várias soluções ou pode ser impossível, onde não há solução. Um sistema linear é um conjunto de equações do primeiro grau, a solução do sistema é onde as retas se cruzam, sendo as retas concorrentes representando um sistema determinado, as retas coincidentes um sistema inderteminado e as retas paralelas representando um sistema impossível.

Sobre o Jogo

Desenvolvimento

O jogo foi desenvolvido digitalmente com foco em acessibilidade e simplicidade. Utilizou-se um design básico e responsivo, pensado especialmente para dispositivos móveis. As cores escuras foram escolhidas para favorecer pessoas com sensibilidade visual. Todo o projeto, desde o design até a codificação, foi concluído em três semanas.

Tecnologias utilizadas

HTML

Hypertext Markup Language (HTML) é a linguagem de marcação padrão para documentos projetados para serem exibidos em um navegador web. Ela define o conteúdo e a estrutura do conteúdo da web. Frequentemente é assistida por tecnologias como Cascading Style Sheets (CSS) e linguagens de script como JavaScript, uma linguagem de programação.

CSS

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilos (cores, fontes, espaçamento, etc.) a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags <style>. Também é possível, adicionar estilos adicionando um link para um arquivo CSS que contém os estilos. Assim, quando se quiser alterar a aparência dos documentos vinculados a este arquivo CSS, basta modifica-lo.

Bootstrap

Bootstrap (anteriormente Twitter Bootstrap) é um framework CSS gratuito e de código aberto direcionado ao desenvolvimento web responsivo e mobile-first front-end. Ele contém HTML, CSS e (opcionalmente) modelos de design baseados em JavaScript para tipografia, formulários, botões, navegação e outros componentes de interface.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. JavaScript permite páginas da Web interativas e, portanto, é uma parte essencial dos aplicativos da web.

jQuery

jQuery é uma biblioteca de código aberto baseada na licença MIT. Possui sintaxe que simplifica: a navegação do documento HTML, a seleção de elementos DOM, criar animações, manipular eventos, desenvolver aplicações AJAX e, criar plugins sobre ela. Tais facilidades permitem a criação de camadas de abstração, para interações de baixo nível de modo simplificado em aplicações web dinâmicas de grande complexidade.
