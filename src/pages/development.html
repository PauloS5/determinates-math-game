O trabalho tem a finalidade de ajudar pessoas a entenderem e resolverem problemas do cotidiano envolvendo determinanes e sistemas lineares, o trabalho foi feito com elementos interativos para que estas pessoas com dificuldade de aprender e fixar bem o conteúdo possam realizar tarefas e trabalhos sobre este tema. Além de ser de cunho educativo, também tem elementos que tornam o jogo divertido de se jogar e aprender, sendo bem simples de resolver os problemas existentes no jogo. Foi também realizado uma responsividade ao site, deixando ele compatível a todos os dispositivos.

O jogo inteiro foi desenvolvido digitalmente, com o objetivo de ficar acessível para que todas as pessoas pudessem jogar, foi utilizado elementos básicos e simples de design para a criação do trabalho, o site foi inteiramente feito para ser responsivo aos dispositivos móveis, onde a maioria das pessoas usam e jogam outros tipos de jogos. O site tem cores pretas para que pessoas com visão sensível tivessem a possibilidade de enxergar melhor os elementos presentes no jogo, foi tomado 3 semanas para a criação do trabalho inteiro, desde os elementos do design, até a própria codificação do próprio.

HTML 5

HTML (HyperText Markup Language) versão 5, é a quinta mais atual versão da linguagem de marcação, o HTML é utilizado para se criar e estruturar páginas web, A linguagem foi desenvolvida com o objetivo de tornar sites e apps mais funcionais e interativos em diferentes dispositivos.

Por que utilizar HTML 5:

  • Linguagem Estruturada: A linguagem permite estruturar conteúdos e organizar partes essenciais para o site, como parágrafos, cabeçalho, rodapé, corpo do site, links, dentre muitos outros. Em resumo, HTML estrutura os conteúdos separadamente, tendo as chamada tags, como blocos de construção.
  • Suporte para vários dispositivos: HTML 5 tem suporte para muitos dispositivos que tenham acesso ao navegador de internet, sendo eles Computadores ou Notebooks, Smartphones, etc. HTML é projetado para ser funcional e compatível com estes dispositivos, assim, podendo funcionar bem em diferentes tipos de tela.
  • Semântica: HTML 5 tem uma semântica rica em suas estruturas, usa tags como "nav", "header", "main" e "footer". Que fazem o código se tornar mais compreensível e bem organizado para desenvolvedores e leitores de código, a semântica, em resumo, é utilizada para não desorganizar o código e estruturar ele de forma rica e compreensiva.

Bootstrap 5

Bootstrap 5 é um framework front-end utilizado para facilitar a criação de sites e designs modernos, sendo um framework da linguagem de estilização Cascading Style Sheet (CSS), Bootstrap oferece uma coleção de classes CSS, componentes prontos e animações JavaScript já programadas, Bootstrap permite desenvolvedores estilizarem seu site e o deixarem responsivos utilizando menos esforço.

Por que se utilizar o Bootstrap 5:

  • Responsividade: O Bootstrap se adapta automaticamente ao layout do site para diferentes tamanhos de tela, sendo móveis ou portáteis, organizando o conteúdo em Linhas e Colunas de maneira mais simples e responsiva.
  • Personalização Simples: Bootstrap 5 utiliza componentes de design prontos e já definidos, como cores, estilos e animações, estes elementos de design são utilizados em Botões, formulários, criação de carroséis, Accordions, menus, etc. Em resumo, Bootstrap utiliza estilos prontos para os elementos HTML, deixando-os com elementos de design.
  • Compatibilidade: Boostrap é utilizado normalmente com os seus designs modernos para ter compatibilidade com muitos dispositivos, funcionando em seus navegadores, adaptando-se automaticamente de acordo com o Layout do dispositivo, elementos que podem mudar de acordo com o tamanho da tela do dispositivo, melhorando a responsibilidade

TypeScript

TypeScript é um conjunto completo da linguagem de programação JavaScript, linguagem criada pela microsoft, que adiciona uma tipagem estática onde o programa tem que definir um tipo de dado: int, string, double, etc. TypeScript também une recursos de Programação orientada a objetos (POO) ao JavaScript tradicional.

Por que utilizar TypeScript:

  • Tipagem Estática: Como mencionado, TypeScript tem uma tipagem estática, onde o programador deve definir o tipo a variável, constante, função ou objetos, o que dá uma "segurança" ao programa, onde se evita possíveis erros de tipagem no código.
  • Compilação para JavaScript: Quando o código TypeScript é concluído, ele, ao ser executado, é convertido ou Transcompilado para JavaScript puro, que pode ser executado em qualquer navegador ou ambiente do Node.js. Isto acontece pois TypeScript não pode ser executado em navegadores ou ambientes como Node.js, eles só podem executar arquivos JavaScript, por isso deve se passar pela Transpilação.
  • Orientação a Objetos: TypeScript tem orientação a objetos, como conceitos de classes, objetos, interfaces, herança, etc. TypeScript tem um suporte a POO mais estruturado do que a linguagem JavaScript, TypeScript apenas reforça e deixa mais robusto ferramentas de Programação orientada a objetos.
  • Escalabilidade: Em geral, TypeScript é utilizado para projetos grandes e complexos, pois facilita a manutenção e organização do código, um código TypeScript pode ser sustentável e resistente a falhas, implementa novas ferramentas, funcionalidades para o sistema e é flexível para manutenção de erros e outros aspectos importantes do Software.

jQuery

jQuery é uma biblioteca JavaScript rápida, pequena e rica em recursos. Ela simplifica tarefas como manipulação do DOM, tratamento de eventos, animações e requisições AJAX, facilitando a criação de funcionalidades interativas com menos código e mais compatibilidade entre navegadores.

Por que utilizar jQuery:

  • Sintaxe Simplificada: jQuery permite fazer ações com poucas linhas de código que normalmente exigiriam mais linhas com JavaScript puro, tornando o desenvolvimento mais ágil e limpo.
  • Compatibilidade entre Navegadores: jQuery resolve diferenças de comportamento entre navegadores automaticamente, oferecendo uma experiência mais uniforme para o usuário.
  • Manipulação de DOM Facilitada: A biblioteca facilita a seleção e manipulação de elementos HTML com comandos intuitivos, como esconder, mostrar, adicionar ou remover classes, elementos, textos, etc.
  • Requisições AJAX: jQuery torna muito mais fácil enviar e receber dados do servidor sem recarregar a página, utilizando funções como $.get(), $.post() e $.ajax().
  • Grande Comunidade e Plugins: jQuery tem uma enorme comunidade e milhares de plugins disponíveis, que ajudam a adicionar funcionalidades como carrosséis, sliders, modais, validação de formulários e muito mais.
